Portfolio Website

Durch meine im Studium, und Arbeit als Werkstudent, erworbene Erfahrung mit HTML, CSS sowie JavaScript fiel meine Wahl bei der Umsetzung einer Portfolio Seite auf das Hosting mit Github. Im Gegensatz zur Verwendung von Wordpress oder anderer CMS hat dies für mich verschiedene Vorteile. Die zwei größten sind volle Anpassbarkeit an meine Bedürnisse sowie die Vermeidung laufender Kosten, der einzige Kostenfaktor der Website liegt im Mieten der Domain.


Zur Farbwahl habe ich mich für ein schlichtes Schwarz, weiß, grau Setup entschieden. Zur besseren Sichtbarkeit der Links sind diese in einem Blauen Ton gehalten. Durch die schlichte Farbwahl und simplen Aufbau liegt der Fokus des Betrachters auf den vorgestellten Projekten.


Über die folgenden Links gelangen sie zum einen zum Source Code, aber auch einem aktuellen Google Analytics Report (erstellt mit Google Looker Studio) sowie einem aktuellen Page Speed Insights Report. Page Speed Insights habe ich dabei vor allem als Anhaltspunkt für Optimierungen der Website genutzt.
